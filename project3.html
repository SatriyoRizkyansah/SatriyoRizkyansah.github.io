Developing Universitas Pamulang - Profile website

By Satriyo Rizkyansah

In my journey as a software developer, one of the projects I am particularly proud of is the development of the Universitas Pamulang (Unpam) Profile website. This project was a collaborative effort, where I worked as a full-stack developer alongside two talented teammates who focused on designing the front-end and assisting with its implementation. The back-end development, however, was entirely my responsibility.

Project Overview

The primary objective of this project was to create a comprehensive and user-friendly profile website for Universitas Pamulang. Using Laravel as the back-end framework and Bootstrap for the front-end design, we built a responsive and visually appealing platform. The website was designed to showcase the university's information, achievements, and offerings in an organized and accessible manner.

My main tasks included developing the back-end functionalities, such as database management, authentication, and ensuring the smooth operation of features like dynamic content updates and secure data handling. Meanwhile, my teammates worked diligently to create an aesthetically pleasing design and user experience for the front-end.
